Frequently Asked Questions

  • Which browser is supported ?
    • The latest or or
  • Why HTML5 ?
    • Because the image files are read and scanned using the HTML5 File API and canvas tag
  • Why does the webcam scanner use the Adobe Flash player ?
    • For browsers not supporting the HTML5 getUserMedia API, images will be captured from the webcam using a flash script
  • When I click on the Webcam, the box below remains white, why ?
    • If getUserMedia API is not supported you have to give access to the Webcam in the Flash settings by right click in the box then Settings and in the Privacy tab select Allow
  • The webcam scanner doesn't recognize the QR Code, why ?
    • The Flash backed image capturing uses small resolution (320x240), you have to focus on the code and have good lighting conditions in order to scan QR Codes
